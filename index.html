Why JavaScript?

JavaScript is flexible!

  • Client, server, mobile, IoT, robots...
  • Write it in any style
  • Large ecosystem
    • Object-oriented or functional Declarative or imperative

JavaScript is growing!

  • Managed by TC39 annual updates
  • Browsers continually add new features
  • Babel turns tomorrow's JS into today's
Next: http://2ality.com/2018/02/ecmascript-2019.html

JavaScript is weird!

  • Continuously evolving
  • Loosely typed
  • No one style guide
  • The DOM is basially a giant global variable

The DOM is weird!

  • DOM = Document Object Model
  • Browsers read HTML tags and turn them into DOM elements
